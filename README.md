# NETHA – AI Analysis of Life Struggles and Resilience

This project analyzes life challenges using data analysis and machine learning.

## Project Overview
NETHA is a storytelling data project that models a girl's life journey through different stages such as school, college, job search, and work environment.

## Dataset
Synthetic dataset with 1200+ rows representing life events.

Features include:

- Age
- Life Stage
- Family Support Level
- Emotional State
- Career Status
- Obstacle Type
- Stress Level
- Outcome

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Machine Learning

Random Forest model used to analyze factors influencing stress levels.

## Insights

- Lower family support correlated with higher stress
- Career interruptions increased emotional stress
- Education stages showed stronger determination
